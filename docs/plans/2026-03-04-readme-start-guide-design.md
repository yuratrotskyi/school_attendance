# Дизайн: README та базова інструкція запуску звітності

## Мета
Створити україномовний README, який дає швидке розуміння проєкту та дозволяє запустити щоденний процес звітності без зайвих уточнень.

## Цільова аудиторія
- Адміністрація школи (директор, заступник).
- Відповідальний за щоденне формування звіту.

## Погоджена структура README
1. Що це за репозиторій.
2. Швидкий старт (5 кроків).
3. Щоденний цикл роботи (06:00-12:00).
4. Структура документації (ключові файли).
5. Правило "втік з уроків" (1+).
6. KPI пілоту на 30 днів.
7. Повна документація (посилання на `docs/`).

## Рішення по базовій інструкції запуску
Окремий файл `docs/ops/attendance/start-guide.md` з коротким покроковим регламентом:
- що підготувати перед першим запуском;
- як запускати щоденно;
- що робити при помилках/неповних даних;
- що перевірити перед відправкою директору.

## Обмеження
- Мова: тільки українська.
- Без нових залежностей і без автоматизації коду на цьому етапі.
- Документаційний реліз без змін бізнес-правил.
