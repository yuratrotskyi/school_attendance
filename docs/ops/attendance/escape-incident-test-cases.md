# Тест-кейси для правила "втік з уроків"

1. Присутній на уроках 1-2, відсутній на 3 => інцидент
2. Присутній на 1, відсутній на 2-3 => інцидент
3. Відсутній з 1-го уроку до кінця дня => не інцидент
4. Присутній на 1-4, відсутній на 5 з кодом EXCUSED_MEDICAL => не інцидент
5. Присутній на 1-3, відсутній на 4 без причини => інцидент
6. Присутній на 1-6 без пропусків => не інцидент
7. Присутній на 1, відсутній на 2 (TECHNICAL_PENDING), після уточнення на UNEXCUSED => інцидент
8. Присутній на 1, відсутній на 2 (TECHNICAL_PENDING), після уточнення на EXCUSED_ADMIN => не інцидент
