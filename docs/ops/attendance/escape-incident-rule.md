# Правило інциденту "втік з уроків"

## Формальне визначення
Інцидент "втік з уроків" виникає, якщо учень має хоча б один (`1+`) пропущений урок після першої зафіксованої присутності у цей навчальний день.

## Не вважається інцидентом
- Учень відсутній з першого уроку і не з'являвся протягом дня.
- На пропущений урок або день є офіційна причина.

## Валідація інцидентів
- Початковий статус: `new`.
- Перевірка класним керівником до 10:00: `in_review`.
- Підсумковий статус до 12:00: `confirmed` або `closed`.

## Правила статусів
- `new -> in_review -> confirmed`
- `new -> in_review -> closed`

Статус `closed` застосовується, якщо підтверджено, що була офіційна причина або помилка внесення даних.
